# YouTube Video Downloader for Android

Приложение для скачивания видео с YouTube в формате MP4.

## 📱 Описание

Простое и удобное Android-приложение, которое позволяет скачивать видео с YouTube в выбранном качестве.

## ✨ Возможности

- 🎥 Скачивание видео с YouTube по ссылке
- 📊 Выбор качества видео (144p, 240p, 360p, 480p, 720p, 1080p и др.)
- 💾 Сохранение в формате MP4
- 🚀 Быстрая загрузка
- 📱 Интуитивно понятный интерфейс

## 🔄 Как использовать

1. **Вставьте ссылку**  
   Скопируйте ссылку на видео с YouTube и вставьте её в поле ввода приложения.

2. **Выберите качество**  
   Выберите желаемое качество видео из доступных вариантов.

3. **Начните загрузку**  
   Нажмите кнопку «Скачать» и дождитесь завершения процесса.

4. **Получите файл**  
   Готовый MP4-файл будет сохранён в памяти вашего устройства.

## 📋 Требования

- Android 5.0 (API level 21) и выше
- Разрешение на доступ к хранилищу
- Подключение к интернету

## ⚙️ Установка

1. Скачайте APK-файл приложения
2. Разрешите установку из неизвестных источников в настройках устройства
3. Установите приложение
4. Запустите и начните использовать

## 🔐 Разрешения

Приложение запрашивает следующие разрешения:
- `INTERNET` — для доступа к сети
- `WRITE_EXTERNAL_STORAGE` — для сохранения видео
- `READ_EXTERNAL_STORAGE` — для доступа к загруженным файлам

## ⚠️ Важная информация

> **Примечание:** Скачивание видео с YouTube может нарушать Условия использования сервиса. Используйте приложение только для загрузки контента, который вы имеете право скачивать (например, видео под лицензией Creative Commons или собственные видео).

## 🛠️ Технологии

- Язык: Kotlin
- Минимальная версия Android: 7.0+ (API 24)
- Формат вывода: MP4
- Архитектура: MVVM с Coroutines
- Библиотеки: OkHttp, Material Design, ViewBinding

## ⚠️ Важное замечание

Данное приложение является **демонстрационным проектом** и содержит базовую структуру Android-приложения для загрузки видео. 

### Для полноценной работы необходимо:

1. **Интеграция с YouTube API** - В текущей версии метод `getVideoUrl()` в классе `DownloadService.kt` содержит заглушку. Для реальной загрузки видео вам потребуется:
   - Использовать официальное [YouTube Data API v3](https://developers.google.com/youtube/v3)
   - Или интегрировать стороннюю библиотеку (например, [NewPipe Extractor](https://github.com/TeamNewPipe/NewPipeExtractor))
   - Или использовать серверный прокси для обработки запросов

2. **Соблюдение условий использования YouTube** - Загрузка видео с YouTube может нарушать [Условия использования сервиса](https://www.youtube.com/t/terms). Убедитесь, что вы:
   - Имеете право на загрузку контента
   - Используете приложение только для личного некоммерческого использования
   - Не нарушаете авторские права

## 🚀 Сборка проекта

1. Откройте проект в Android Studio
2. Дождитесь синхронизации Gradle
3. Подключите устройство или запустите эмулятор
4. Нажмите Run (Shift+F10)

Или через командную строку:
```bash
./gradlew assembleDebug
```

APK файл будет создан в `app/build/outputs/apk/debug/`
